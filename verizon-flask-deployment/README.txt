Follow the steps to start the app and send requests:

1) install required modules utilizing "requirements.txt" in your virtual environment

2) run app.py, app will run on http://127.0.0.1:5000/ --> this will use model.py

3) run dash2predict.py, this will run on http://127.0.0.1:1000/ --> send requests to app

4) upload input csv file and get predictions accordingly
